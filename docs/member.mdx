# 排序规则及注意事项

1. 优先展示 GitHub ID 和 blog 都有的同学
2. 如果都只有一个，优先展示有 GitHub ID 的同学，再展示有 blog 的同学
3. 如果都没有，按姓名字典序排列

:::warning

比较 GitHub id 和 blog 时也按字典序排列
:::

:::tip
以 2023 为例，2023 包含 2022-2023 学年成功加入工作室的同学，既包含 23 年 9 月份统一加入（即通过一整年考核），也包括这一年间提前加入的同学
:::
